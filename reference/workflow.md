# 工作流 · 端到端执行

> 本篇管"怎么做"；判据见 `framework.md`。所有数据来自 Longbridge OpenAPI MCP（工具名以你的 MCP 配置为准）。
> 贯穿纪律：**禁止在完成盲扫前先看自己的持仓/昨日观点（防锚定）；强制扫"主线之外"；每个判断带证伪预算；矩阵决定输出，不凑认购/认沽对称。**

## 阶段0 · 盲扫全市场（锁异常）
- 工具：市场状态/温度（`marketStatus` / `marketTemperature` / `historyMarketTemperature`）、三大指数 `quote`（HSI/HSTECH/HSCEI）、`industryRank` **全表**、`rankList`（热门交易/热度上升）、`topMovers`、`anomaly`、`screenerSearch`（粗估涨跌家数=广度）。
- 产出：当日 Top 异常清单（广度、领涨/领跌板块、资金落点）。

## 阶段1 · 收窄到可打票池
- 剔除仙股/微盘（无权证）→ 只留**有权证的流动大中盘**。
- 提醒：`topMovers` 按波幅排出的多是仙股，对权证无用；用 `rankList`（热门交易）+ 成交额/净流入定真正活跃的大中盘。

## 阶段2 · 双向鱼身定位（核心）
- 多周期轨迹（5/10/20 日 + YTD + 半年）初判位置；
- `candlesticks` 验真突破/破位（注意时间戳偏移，用 prev_close 校验）；
- **对每个候选（含"看起来涨/跌多"的）拉 `capitalDistribution` 看大/中/散结构** → 定鱼段；
- **防错杀闸门**：鱼尾必须有"资金结构翻转 + 抛物线速度"双证，不许按涨跌幅判；有疑必拉数据。
- 产出：认购×{鱼头/鱼中}、认沽×{鱼头/鱼中} 候选；鱼尾出清，但记下"反向潜在鱼头"（高位大单派发→认沽鱼头；低位大单吸筹→认购鱼头）。

## 阶段3 · 权证结构筛选（第二道闸）
- 工具：`warrantList`（按 IV 升序；另拉对应牛/熊证 Bull/Bear）、`calcIndexes` 补 Premium/Delta/EffLev、`staticInfo` 取每手股数。
- 闸门（framework L7/L8）：IV 非板块极值、到期 ≥3 月、街货 <50%、价差可接受；**股票鱼身与权证经济性两闸全过**。
- 产出：每标的 1–3 只候选轮（价内/平价/价外 或 牛熊证）。

## 阶段4 · 量化盈亏比 × 持仓周期 + 排雷
- 定 base/目标/止损位 → 用实际杠杆算各情景回报与盈亏比（标注线性近似；凸性使顺势一侧更好；隐含"到止损就走"）。
- 持仓周期分档（按需，例如 短打 / 波段 / 偏长）：短打优先牛熊证/价内（低 theta）；波段用平价/价外（凸性）；偏长扣 theta 估算，价外不宜放太久。
- 排雷：IV 极值、街货过高、流动性薄、收回价过近、无轮。

## 阶段5 · 执行前实时确认
- 工具：`warrantQuote`（live 价/IV/delta）、`depth`（买卖价差+深度）、`capitalFlow`（live 净流向）+ `capitalDistribution`（结构）。
- 闸门：价差可接受（penny 轮看 tick 数与发行商深度）；**资金结构印证方向**（确认或证伪，主力=参考非裁决）；**已确认即动，未确认挂条件单**。
- 警惕：认购权证在突破日往往已大涨（杠杆吃掉日内涨幅），别在尖顶 tick 市价追，用限价。

## 阶段6 · 出行动盘
- 字段：**标的 × 鱼段 × 工具 × 盈亏比 × 周期 × 触发 × 止损 × 仓位**（鱼头小 / 鱼中核心）。
- 矩阵决定输出：**某侧全是鱼尾就空仓等触发，不硬做对称。**

## 可选 · 下单与监控（若 MCP 开通交易，建议模拟盘）
- **下单前强制核账户**：`accountBalance` + `stockPositions` 确认是预期账户（尤其模拟盘），再 `submitOrder`。
- 止损/止盈若为"收盘价/资金流"条件，挂不成普通委托 → 用定时巡检 + 券商到价提醒（`alertAdd`）兜底。
- 监控频率随持仓周期定：波段持仓约每交易日 2–3 次（开盘后早预警 / 午后止盈 / 盘前收盘决定性判定）。

## 示例（一次完整跑动的形态，仅说明流程，非推荐）
盲扫 → 锁定"窄市、半导体领涨、某医药板块首日轮动" → 双向鱼身：认购鱼中=领涨代工龙头（突破+大单吸筹）、认购鱼头=刚启动的医药轮动龙头；认沽侧若超跌大票经资金结构复核为"低位吸筹"=非认沽（潜在认购鱼头）、高位龙头大单派发=潜在认沽鱼头 → 筛轮（剔除 IV 极值/无轮标的）→ 量化盈亏比×周期 → 盘口确认 → 出行动盘（认购两腿可执行、认沽空仓等破位触发）。
