# 框架 · 权证盈亏比鱼身定位（理念与判据）

> 多空双向。本篇管"怎么想"；执行步骤见 `workflow.md`。所有数据来自 Longbridge OpenAPI MCP。

## L0 内核
权证无独立价值，**盈亏比是合成的**。脱离方向，期望值为负（时间损耗 + 买卖价差是确定性流出）。
真实分解：`ΔW/W ≈ 实际杠杆×ΔS/S + vega×ΔIV − theta×Δt − 价差`。
主路径：**先定方向 → 鱼身定位 → 资金印证 → 选轮 → 量盈亏比 → 盘口执行**（方向驱动，非相对价值套利）。

## L1 盈亏比的真相
- **杠杆等比放大盈亏两端，不改善盈亏比。**
- 真正制造不对称的是三样：① 凸性（gamma，平价/价外最强）；② 标的目标/止损距离比；③ 严守止损。
- 一切盈亏比数字都隐含前提：**到止损价就走，不持有到归零。**

## L2 鱼身定位（组织全局的尺，双向对称）

| | 认购健康 | 认沽健康 | 鱼尾（禁） |
|---|---|---|---|
| 位置 | 突破后中段 | 破位后中段 | 抛物线顶 / 瀑布底 |
| 大单 | 净进（吸筹） | 净出（派发） | **结构翻转** |
| 散户 | 净出 | 净进 | **结构翻转** |
| 距历史价位 | 无关 | 无关 | 无关 |

- **鱼头** = 刚拐 · 轻仓 · 限亏工具（价外证）；
- **鱼中** = 确认 · 核心仓 · 高效工具（平价证 / 牛熊证）；
- **鱼尾** = 一票否决。

## L3 鱼尾判据：结构 + 速度，不是幅度
- "跌太多"之于认沽 = "涨太多"之于认购 = **同一个最贵的本能**。下跌趋势天生不断创相对新低，那正是它该干的事。
- 鱼尾 = **资金结构翻转（派发/吸筹反向）+ 抛物线速度**，二者需双证。
- **绝不许按涨跌幅枪毙。错杀 = 框架有洞，必须补。**

## L4 资金结构：主力是参考，不是裁决
- 结构性质：吸筹 = 大进散出；派发 = 大出散进；鱼尾 = 结构翻转。
- **但主力会错、会诱多诱空、大单也可能只是发行商对冲。** 资金的职责是"确认或证伪我的方向判断"，**不替我下结论**；再强的单日大单也不是保票。
- 用法：方向判断在前，资金验证在后；资金与价格/位置矛盾时是"警示"，不是"反向信号"。

## L5 进出场纪律
- **未确认才等，已确认就动**（突破/破位 + 当日资金结构对齐 = 足够进场，不必二次确认）。
- **犹豫放在离场端，不放进场端。**"别追"≠"等更便宜的价"，而 = 现在进 · 对的仓位 · 证明后加 · 限价别扫尖顶 · 不在一天大的趋势上 All-in。
- **跟主力 = 跟进也跟出**；信心是从资金流租来的 → 结构一翻转立即走。

## L6 双向执行不对称
方向对称，执行不对称：**认沽止损更紧、仓位更小**（下跌反抽与轧空比上涨凶；短覆盖无消息也能机械引爆）。

## L7 两道独立闸门
**"股票鱼身定位"与"权证经济性"分开判，不可合并。**
- 股票可以是鱼中（被吸筹），但其权证可能不经济（IV 极值 / 回本过远 / 杠杆被压）→ 仍不交易。
- 反之权证再便宜，股票若是鱼尾 → 也不做。**两闸全过才出手。**

## L8 权证经济性判据（第二道闸）
IV（同标的横比、避板块极值）· 实际杠杆 · 溢价/回本点 · 价内外+delta（价内高胜率/价外高凸性）· **到期 ≥3 月**（避 theta 悬崖）· 街货 <50% · 发行商价差 · 换股比。
- 牛熊证（CBBC）：无 theta/IV 但有强制收回，**收回价须越过止损位**。
- 工具配鱼身：鱼头 → 价外限亏；鱼中 → 平价证 / 牛熊证。
- 工具配周期：短打 → 牛熊证/价内（低 theta）；波段 → 平价/价外（凸性）；偏长 → 扣 theta、方向慢则用 CBBC。

## L9 数据现实（Longbridge MCP）
- 不直接给权证 theta（需用 IV + 到期估算）；
- K 线时间戳可能偏移 +1 日（读某日收盘取前一日历日那根，用 prev_close 交叉校验）；
- 权证只存在于流动大中盘（仙股异动无关）；
- **部分标的无轮 → 不可执行即出局**。
